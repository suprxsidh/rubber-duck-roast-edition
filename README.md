# 🦆 Rubber Duck Debugging: The Roast Edition

A roguelike game where you play as a rubber duck fighting bugs in codebases, with an AI roast commentator that roasts your every move!

## How to Run

```bash
cd rubber_duck_roguelike
./venv/bin/python main.py
```

## Controls

- `1-5`: Select attack
- `U`: Open upgrade shop
- `H`: Heal (30 gold)
- `ENTER`: End turn / Start game

## Features

- 🦆 Play as a rubber duck
- 🔥 3 AI roast personalities: Socratic, Existential, Corporate
- ⚔️ Turn-based combat with 2 actions per turn
- 👹 5 enemy types + Boss fights
- 🛒 Upgrade shop
- 💰 Gold/XP/HP systems
- 🎯 High score tracking

## The Roast System

The AI roast commentator roasts you based on:
- Your attacks
- Mistakes
- Damage taken
- Victories
- Defeats
- Upgrades

Each new game randomizes the roast personality!
